# Selenium Automações

## Conceito Geral
- O selenium é uma biblioteca de software livre que ajuda a resolver trabalho manuais e repetitivos utilizando o browser;
- Foi desenvolvido e pensado para automação de testes de software e pode ser utilizado para criação de bots;
- O selenium é utilizado para web scraping, ou seja, raspagem de dados da internet

## Selenium Web Driver e Selenium Grid
### Web Driver
- O web driver é responsável pela manipulação do navegador de forma nativa, ou seja, ele opera o navegador como um usuário faria, de maneira local ou remota.
- Quando falamos do Selenium Web Driver, estamos falando da lingaugem e as implemnetaçĩes de do código de controle do navegador individual.
